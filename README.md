## Setup & Deployment (Google Colab)
1. Open the `TTF9_Universal_Auditor_Public.ipynb` in Google Colab.
2. Run the cell to install system dependencies (Tesseract, Poppler).
3. The script will request Google Drive access to construct the `TTF9_System` folder hierarchy.
4. Input your Groq API Key when prompted securely.
5. Upload your target document and let the Active Inference loop process the data.
